# DSA-Java

A collection of fundamental queue data structure implementations in Java, including linear and circular queues, with both direct and stack-based algorithmic approaches. Ideal for learning and practicing core data structures and algorithms.

## Contents

- **LinearQueue.java:** Standard linear queue implementation.
- **CircularQueue.java:** Efficient circular queue design.
- **LinearQueueUsingStacks.java:** Implements a linear queue using two stacks.
- **CircularQueueUsingStacks.java:** Implements a circular queue using stacks.

## Features

- Pure Java implementations for educational and interview prep.
- Demonstrates multiple ways to solve typical queue-related problems.
- Modular and easy to extend for further algorithmic experimentation.

## Usage

Clone the repository and use the Java files directly in your projects or studies:

```sh
git clone https://github.com/natnaeleyuel/DSA-Java.git
```

Compile and run the desired file using:

```sh
javac [FileName].java
java [ClassName]
```
